# Personal Book Manager

A MERN stack application to manage personal reading lists.

## Tech Stack
- Next.js
- Node.js
- Express
- MongoDB
- JWT Authentication
- Tailwind CSS

## Features
- User authentication (signup/login/logout)
- Add, edit, delete books
- Reading status tracking
- Filter books by status
- Inline editing

## Setup

### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm run dev
